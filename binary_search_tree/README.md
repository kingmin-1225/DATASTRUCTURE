# Binary Search Tree
- 이진트리를 활용하여 정렬상태를 유지하는 트리     
- 힙과 다르게 항상 `left_child` < `parent` < `right_child` 의 순서로 값이 정렬된 채로 유지됨(중위순회할 경우 정렬된 리스트 출력)

## 삽입 알고리즘
기준 노드와 넣고자 하는 정수의 크기를 비교해서 크면 `right_child`를 기준으로 작다면 `left_child`를 기준으로 재귀 호출
만약 기준 노드가 `None`이라면 그 위치에 새로운 노드를 삽입

## 삭제 알고리즘
- 자식노드가 1개 이하라면 자기 위치를 자식노드로 대체
- 자식노드가 2개일 때는 왼쪽 자식노드를 루트로하는 서브트리의 가장 큰 값을 자기 위치로 옮기고 찾아낸 가장 큰 값의 자식노드는 반드시 1개 이하이기 때문에 위에 자식노드가 1개 이하일 경우의 알고리즘에 따라 처리해줌

## Methods
- `_insert(node, e)` : node 기준으로 탐색 후 e 삽입
- `_remove(node, e)` : node 기준으로 탐색 후 e 삭제
- `_inorder(node)` : node를 기준으로 중위순회
- `add(e)` : e 삽입
- `delete(e)` : e 삭제
- `print_nodes()` : 트리 값들 출력
